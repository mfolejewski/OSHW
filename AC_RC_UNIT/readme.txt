Quite easy, evaluation board for Bluetooth 5.0 modules.

Used components:
- Bluetooth 5.x module (BLE) - Fanstel & Laird compatible (Nordic nRF528xx)
- SWD interface
- RGB LED
- 16-button keyboard
- buzzer
- pin headers
- DC/DC converter + 3V lithium battery
- 128x64 COG LCD mono
- I2C interface

PCB: 4-layer, standard stackup.
