OSHW PCB PROJECT #03: LED Matrix display 8 x 16 pix, VER1/REV1

Main specification:
- resolution: 8 x 16 pix (128 LEDs), monochrome (RED color),
- LED control: full static, drivers LED with current stabilisation,
- signal connectors: standard IDC10 connectors,
- power source for logic circuits and for LED diodes: 5VDC,
- used 0805 and 1206 chip passive components,
- LED components: standard, oval, high brightness, 2.54 pitch,
- PCB size: 474.75 x 224.75 mm,
- 2-layer PCB,
- used 128 LED pixels.

Only for non-commercial use!
